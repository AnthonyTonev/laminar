Class KafkaConsumerService<TValue>

A class that wraps kafkajs Consumer and implements laminar Service, so lamniar can handle its lifecycle. When service service starts, subscribe to the topic and call kafkajs consumer run method.

On each message / each batch, decode the message using schemaRegistry and pass it as decodedValue next to the value param.
