Class PgClient

A class that wraps a PoolClient, cathing errors and re-throwing them with more diagnostic information.

const pool = new Pool({ connectionString: '...'});
const client = await pool.connect();

const pgClient = new PgClient(client, console);

await pgClient.query('...');

transaction

  • transaction<T>(operation: (db: PgClient) => Promise<T>): Promise<T>
    • Defined in packages/laminar-pg/src/pg.client.ts:86

    Wrap the operations in a transaction, commit it when the Promise is resolved and rollback if there is an exception. Supports nested transactions, but only the outermost transaction is used. The result of a transaction is the return of its "operations" function.

    const result = await db.transaction(await (tr) => {
      await tr.query('...');
      await tr.query('...');
      return 'value';
    });
