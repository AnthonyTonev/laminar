Interface Response<Content, Status>

A strongly typed response object. You can use the various response helpers to produce / modfiy those objects.

For example those helpers would return a response type object: response, jsonOk, jsonNotFound, htmlOk

Type parameters

  • Content

    The strict type of the content. Used by type generators to specify the required body type as a type literal

  • Status

    The strict status type. Used by type generators to specify the status as a type literal, for example 302 | 200
